# Sosial Media Unusia
> Semangatt

## Instalasi Sosial Media Unusia (Laravel)
- composer install,
- copy file .env.example to .env,
- php artisan key:generate,
- Jalankan MySQL di XAMPP,
- Akses database
- Buat database kreasikomunitas, dan
- php artisan migrate atau import kk.sql
- php artisan serve
